Ideas from websites


While making this website, I got a few ideas from various other websites and implemented those ideas in this website.




Transparent Navigation Bar


One of the first ideas I had in mind while doing this website is this one. I stumbled on this website while shopping for clothes and I immediately liked the transparent navigation bar and it has stuck on my mind since.

To implement this what I did was I made a header section and put a background image on it using css [background-image: url(../images/bgresearch.jpg);]. Then I kept the navigation bar and all it's items on top of the background image using unordered list, hover, anchor tags and other HTML tags and CSS.


Visit the website: State of Mind Clothing



Product Specifications


While I was browsing through some apple products in the Evo Store webpage, the specifications and product display was really minimalistic and pleasing. I liked this feature because of it's simplicity and functionality as displaying the specs in a table wise format might prove really efficient

To implement this feature, all it needed importantly was the table tag. I set the table border to transparent and set the th, td and tw accordingly.


Visit the website: Evo Store


Google Maps


When I was searching for colleges after my A-Levels, I came upon Islington College's website. I wanted to physically come and enquire about the college and clicked on the contact page. There I found the external link to Google Maps. Ironically, that's how I knew the location of the college and hence joined Islington.

For this, I simply went to Google Maps, searched for my location and embedded the map. After copying the given HTML tag from there, I pasted it on my html file.


Visit the website: Islington College


Customer Review


In this day and age, e-commerce is used by almost every household. And in doing so, we can see many customer reviews in many products. If there is one e-commerce platform all Nepalese people use, it's Daraz. I've used it myself, and while using Daraz, I've seen many customer reviews. So, while I was making this website, I used this feature from Daraz, but I could've referenced any other platform as almost all platforms have customer reviews.

To use this feature, i used a section tag. Used a division class to set up a column. Then, to make a card-like textbox, I used another division tag and gave it a background color. I added some stars to it using icon tag and simply repeated the textbox process as required.


Visit the website: Daraz
